PIAIC Offers the following courses

  1. AI
  2. CNC
  3. BCC
  4. IOT

Artificial Intelligence

The Artificial Intelligence program of the PIAIC takes the student through the foundation of AI technology. The technologies Include: Python, Numpy, Pandas, Tensorflow 2.0 & flask.




Cloud Native Computing

The Cloud Native Program prepares the students to program in open source technologies like Linux, Docker & Kubernetes, Node.js, JavaScript React & React Native.




BlockChain Computing

The main emphasis of Block Chain Computing is to understand the blockchain like Etherium and Hyperledger (Fabric) and learn to program in languages such as Solidity, JavaScript on framewors like Node.js.




Internet of Things

The internet of things program introduces the sutdents to open source development technologies like Rust, Embedded Rust, WebAssembly, Hyper, Diesel, Google Assistant, and Kubernetes while making them efficient in developing Microservices IOT Edge.
